# QuantLab Roadmap

## Current Sequence

1. Phase 3B - BTC 10-event multi-event content review.
2. Phase 3C - Metric policy + event inventory scope decision.
3. Phase 3D - Multi-symbol event inventory / coverage preview.
4. Phase 4A - Decision-time label dataset.
5. Phase 4B - Decision-time feature dataset.
6. Phase 4C - Rule baseline + ML benchmark.
7. Phase 5A - Execution simulator / executable backtest.
8. Phase 5B - Shadow live signal generation.
9. Phase 6 - Controlled small-capital production.

## Current Boundary

Phase 3B is the immediate next phase. It reviews accepted Phase 3A BTC 10-event
outputs and recommends metric policy. It does not implement Phase 4A labels,
Phase 4B features, ML training, live trading, order execution, or production
promotion.

## Direction

The Event Scanner remains a research/data-discovery layer. Later trading
research moves toward decision-time labels/features, positive expected value
after fees/spread/slippage/latency, executable backtesting, shadow live signal
generation, and only then controlled production.
